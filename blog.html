Important FAQ

What is the Document Object Model (DOM)?

DOM is a programming interface for html document

How do you select an element from the DOM?

There are many ways for select an element form the DOM. (i.) getElementById() (ii.) getElementsByTagName() (iii.) getElementsByClassName (iv.) querrySelector() and (v.) querrySelectorAll().

What is event delegation in the context of DOM, and why is it useful?

When a eventListener add in a parent element instead of all children element that is called event delegation. By event delegation we use addEventListener only once not eventListener add in all children element.

How do you manipulate an element's attributes and style using the DOM?

By using setAttribute and getAttribute we can easily manipulate an element's attributes. For example: document.getAttribute('something').style.color = 'red'.
